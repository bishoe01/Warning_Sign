# Roadmap: AI 근로계약서 도우미 MVP

> 상태: DRAFT  
> 기준일: 2026-05-31  
> 원칙: 화면 → 이미지 → 서버 → OCR → AI → 결과 → 제출물 순서로 붙인다.

## Phase 0. 기준 고정

목표: 개발 중 흔들리지 않게 범위와 샘플을 고정한다.

| 작업 | 완료 기준 | 상태 |
|---|---|---|
| 대상 사용자 고정 | 외국인 근로자 방향으로 PRD와 AGENTS 정리 | done |
| 샘플 계약서 확보 | 표준근로계약서 또는 데모용 샘플 1장 선정 | todo |
| 정답 JSON 작성 | 사람이 직접 핵심 조건과 주의 조항 샘플 응답 작성 | todo |
| OCR API 선택 | CLOVA OCR 또는 Google Vision 중 하나로 결정 | todo |

## 줄이는 순서

시간이 부족하면 아래 순서로 줄인다.

1. 카메라 촬영을 빼고 갤러리 이미지 선택만 남긴다.
2. 베트남어 설명을 후순위로 두고 한국어+영어부터 완성한다.
3. APK 빌드를 뒤로 미루고 Expo 실행 영상으로 보완한다.
4. 실제 OCR 장애 대비로 샘플 모드를 발표 hero 흐름으로 사용한다.
5. AI 분석이 불안정하면 샘플 계약서에 대한 고정 JSON으로 데모를 완주한다.
