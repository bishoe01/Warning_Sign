# Phase 9 다국어 + 마이페이지 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 사용자가 고른 언어가 앱 전체 UI와 분석 결과에 함께 적용되도록 구현한다.

**Architecture:** 모바일은 `src/i18n`에서 언어 메타데이터, UI 문구, localized value helper를 관리하고 `settingsStore`가 선택 언어를 AsyncStorage에 저장한다. 서버는 `language` 파라미터를 16개 언어로 확장하고, 분석 결과를 `Record<language, string>` 형태로 반환한다.

**Tech Stack:** React Native + Expo Router + AsyncStorage, FastAPI + Pydantic + OpenAI JSON mode.
