## 책 메모
- expo-file-system SDK56 신규 API는 동기/비동기가 공존한다: 파일 복사는 **`copySync`**(동기), `copy`는 Promise 반환(비동기). 플랜이 `copy`를 동기처럼 `forEach`에서 await 없이 쓰면 floating promise인데 **tsc는 이를 못 잡는다**(eslint 룰). 설치된 `.d.ts`를 직접 읽어 시그니처를 검증한 게 주효했다.
- expo-router `typedRoutes: true`면 새 라우트 추가 시 `.expo/types/router.d.ts`(gitignore)가 재생성돼야 `router.push('/history')`가 타입체크된다. tsc만으론 안 되고 `expo export`/dev 서버가 생성한다. 임시 수동 패치로 진행 → export가 정식 재생성.
- 서브에이전트 스폰도 세션 토큰 한도를 공유한다(Task 11에서 0토큰 스폰 실패). 한도 임박 시 컨트롤러가 직접 구현으로 전환하면 된다.
- 코드 리뷰는 맹종하지 말 것: RN 리스폰더 시스템상 중첩 Pressable의 내부 버튼은 양 플랫폼에서 정상 동작(리뷰어 주장은 오류), `&apos;`도 babel이 컴파일 타임에 `'`로 디코딩(렌더러 무관) — 둘 다 근거로 기각/정리했다.
