## 책 메모
- OCR 출처 UX는 AI 설명을 더 예쁘게 보여주는 기능이 아니라, 사용자가 원문에서 직접 검증하게 하는 신뢰 장치다. 매칭이 약하면 하이라이트를 억지로 만들지 않는 쪽이 맞다.
- 공모전에서는 "AI가 계약서를 분석했다"보다 "사용자가 AI 분석 근거를 사진 원문에서 직접 확인할 수 있게 했다"가 더 설득력 있는 차별점이다.
- 커밋 전에는 staged snapshot을 그대로 믿지 말고, 이전 세션 변경과 이번 세션 변경을 분리해 다시 stage한다.
